Плагин закрывает ссылки на сайты комментаторов и убирает replytocom.

== Description ==

Плагин закрывает ссылки на сайты комментаторов, оставляя возможность перехода на эти сайты.

Кроме того, плагин убирает replytocom из ссылки ответа на комментарий.

Закрытие ссылок осуществляется путём подмены тегов "`<a>`" на "`<span>`", удаления атрибутов "href" и идентификатора "http://" из строки адреса. 

Таким образом ссылки не распознаются поисковыми системами, а их кликабельность и переход на сайт комментатора решаются при помощи css и специального скрипта.

Более подробную информацию вы найдете на [странице плагина](https://obg.kz/zakryt-ssylki-v-kommentariyax-wordpress.html).

Если вам понравился мой плагин, то `пожалуйста` поставьте ему 5 звезд.

== Installation ==

1. Загрузите плагин ark-hidecommentlinks на свой блог
2. Активируйте его
3. Пользуйтесь.
4. Более подробную информацию о плагине Вы можете найти на [странице плагина](https://obg.kz/zakryt-ssylki-v-kommentariyax-wordpress.html).
